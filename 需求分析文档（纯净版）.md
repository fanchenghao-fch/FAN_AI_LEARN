## 1. 产品定义

### 1.1 一句话描述

> **把任何知识，变成一场你停不下来的闯关游戏。**

### 1.2 核心价值主张

| 痛点 | 解决方案 |
|------|---------|
| 学知识枯燥，难以坚持 | 闯关游戏化，即时反馈，让学习像刷短视频一样上瘾 |
| 找学习资料费时费力 | AI 一键生成题目，无需自己整理 |
| 学完就忘，缺少复盘 | 自动生成知识总结 + 智能错题本 + 遗忘曲线提醒 |
| 学习孤独，缺少动力 | 好友PK、排行榜、分享报告、学习圈子 |

### 1.3 产品愿景

成为**知识学习的 "AI 引擎"** —— 无论你想学什么，只需输入一句话 / 一个文档 / 一个链接，AI 就能自动生成一套高质量的交互式学习体验。覆盖 K12 学科、大学课程、职业技能、考公考研、企业培训、兴趣爱好等全部学习场景。

## 2. 核心功能需求（MVP）

### 2.1 功能全景

```
┌─────────────────────────────────────────────────────┐
│                    MVP 功能模块                        │
├─────────────┬─────────────┬─────────────┬─────────────┤
│  知识输入    │  AI 出题     │  闯关答题    │  学习复盘    │
├─────────────┼─────────────┼─────────────┼─────────────┤
│ · 文本输入   │ · 智能搜索   │ · 选择题     │ · 成绩单     │
│ · 文档上传   │ · 题目生成   │ · 判断题     │ · 错题本     │
│ · 链接解析   │ · 答案验证   │ · 即时反馈   │ · 知识总结   │
│ · 语音输入   │ · 知识溯源   │ · 知识讲解   │ · 分享海报   │
├─────────────┴─────────────┴─────────────┴─────────────┤
│                    用户系统 + 历史记录                   │
└─────────────────────────────────────────────────────┘
```

### 2.2 知识输入模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| **文字输入** | 用户输入一句话/一段话描述想学的知识。例如："我想学习中国近代史的重要事件" | P0 |
| **文档上传** | 支持 PDF、Word、PPT、TXT，AI 解析内容后出题。上限 10MB | P0 |
| **链接解析** | 粘贴网页 URL / 公众号文章链接，AI 抓取内容后出题 | P1 |
| **图片识别** | 拍照/截图上传（如课本页、笔记），OCR 识别后出题 | P1 |
| **视频解析** | 上传视频/粘贴视频链接，AI 提取字幕/语音转文字后出题 | P2 |
| **语音输入** | 直接说出想学的知识，语音转文字后出题（尤其适合老年用户） | P2 |

### 2.3 AI 出题模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| **联网搜索** | 当用户输入的知识点不够具体时，AI 自动搜索最新相关内容扩充知识库 | P0 |
| **题目生成** | 基于知识内容自动生成 N 道选择题/判断题，每道题 4 个选项 | P0 |
| **难度分级** | 自动评估题目难度（简单/中等/困难），或允许用户指定难度 | P0 |
| **题目数量** | 默认 10 题，用户可自定义 5-50 题 | P0 |
| **题型混合** | 选择题为主 + 判断题 + 填空题（P2） | P0 |
| **答案校验** | AI 自我验证正确答案的准确性，标注知识来源 | P1 |
| **图片生成** | 为适合配图的题目（如背单词、识图）自动生成配图 | P2 |
| **题目预览** | 出题后允许用户预览和手动调整（删除/修改某题）再开始闯关 | P1 |

### 2.4 闯关答题模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| **闯关流程** | 逐题展示，选择答案后立即显示正误 | P0 |
| **即时反馈** | ✅ 答对：绿色动画 + 鼓励语 + 知识要点回顾 | P0 |
| **即时讲解** | ❌ 答错：红色标记 + 正确答案高亮 + 详细知识讲解 | P0 |
| **进度条** | 实时显示当前进度（如 3/10）和正确率 | P0 |
| **计时器** | 可选：每题限时作答（增加紧迫感），也可不限时 | P1 |
| **跳题/回看** | 允许跳过不确定的题，最后统一回看；也支持回到上一题 | P1 |
| **音效反馈** | 答对/答错不同音效，增强游戏感 | P1 |
| **连击系统** | 连续答对有 combo 特效（如 🔥连击×5），激励用户 | P1 |
| **生命值** | 可选模式：总共 N 条命，答错扣命，0 命通关失败（增加挑战性） | P2 |
| **AI 追问** | 答错后 AI 生成一道类似题追问，确保真正掌握 | P2 |

### 2.5 学习复盘模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| **通关结果** | 展示得分、正确率、用时、排名（如有排行榜） | P0 |
| **知识总结** | AI 生成该知识领域的核心要点总结（3-5 条精华） | P0 |
| **错题回顾** | 列出所有答错的题 + 正确答案 + 详细解析 | P0 |
| **掌握度雷达图** | 可视化展示知识子领域的掌握情况 | P1 |
| **分享海报** | 自动生成精美海报（含分数、知识领域、一句金句），可分享到朋友圈/群 | P0 |
| **学习建议** | AI 根据答题表现，推荐下一步学习方向 | P1 |

### 2.6 用户系统

| 功能点 | 描述 |
|--------|------|
| **微信一键登录** | 通过 `wx.login` 获取用户身份，无需额外注册 |
| **学习历史** | 记录所有闯关记录：时间、知识领域、题目数、得分 |
| **我的错题本** | 聚合所有错题，支持按知识领域筛选和重新作答 |
| **收藏/书签** | 收藏感兴趣的闯关游戏，方便回看 |

## 3. 扩展功能需求

### 3.1 V1.1 — 知识库管理（B端价值核心）

| 功能点 | 描述 |
|--------|------|
| **创建知识库** | 用户（企业/个人）可创建专属知识库，上传多个文档作为出题素材 |
| **知识库管理** | 增删改查文档，管理知识库成员（企业场景） |
| **定向出题** | 从指定知识库中出题，而非全网搜索 |
| **题库沉淀** | 生成的题目可保存到题库，反复使用 |
| **考试模式** | 限时考试模式（区别于轻松闯关），适合企业培训考核 |

### 3.2 V1.2 — AI 生图 & 多媒体

| 功能点 | 描述 |
|--------|------|
| **题目配图** | AI 根据题目内容自动生成配图（如单词对应的实物图） |
| **看图答题** | 看图选择题（如 "图中是什么植物？"） |
| **语音读题** | TTS 朗读题目和选项，适合老年用户和通勤场景 |
| **数字人老师** | AI 数字人出镜读题 + 讲解，增强教学临场感 |

### 3.3 V1.3 — 社交 & 游戏化

| 功能点 | 描述 |
|--------|------|
| **多人对战 PK** | 实时/异步 PK，同领域同难度，比正确率和速度 |
| **排行榜** | 全站排行 / 好友排行 / 领域排行（周/月） |
| **成就系统** | 徽章收集（学习达人、百题斩、十连胜等） |
| **每日挑战** | 每日一道 AI 精选趣味题，培养学习习惯 |
| **学习打卡** | 连续学习天数记录，支持订阅消息提醒 |
| **学习圈子** | 按知识领域建圈子（考研圈、Python圈、历史圈）共享题库 |

### 3.4 V1.4 — AI 导师模式

| 功能点 | 描述 |
|--------|------|
| **AI 追问** | 答错后 AI 追问一道相关知识题，确保举一反三 |
| **学习路径** | AI 根据用户知识水平，规划个性化的学习路线图 |
| **智能复习** | 基于艾宾浩斯遗忘曲线，在最佳时间点推送复习提醒 |
| **AI 对话学习** | 非闯关模式——以 AI 对话的方式学习知识，AI 边讲边提问 |

## 4. 亮点功能创意

> 以下是差异化亮点功能，可以作为产品出圈的核心竞争力。

### 🌟 亮点 1：「AI 学习伴侣」— 拟人化学习伙伴

**概念**：不是冷冰冰的答题页面，而是一个有性格的 AI 学习伙伴陪你闯关。

- 用户可选择不同的 AI 伙伴形象（学霸猫、智慧老人、书虫机器人等）
- AI 伙伴有情绪反馈：答对时会兴奋，答错时会鼓励，连击时会惊叹
- AI 伙伴会记住用户的学习风格，如"你上次在这个知识点上卡住了，我再帮你巩固一下"
- **走红潜力**：萌宠/角色 IP 自带传播属性，截图分享欲强

### 🌟 亮点 2：「知识对战」— 微信好友实时 PK

**概念**：直接利用微信社交关系链，发起知识挑战。

- 发起人选定知识领域 → 生成对战题目库 → 微信分享邀请卡片 → 好友点击进入对战
- 同屏实时 PK（类似头脑王者），看谁正确率高
- 对战结束生成双方对比报告，可分享到朋友圈"叫板"
- **走红潜力**：社交裂变的天然引擎

### 🌟 亮点 3：「一分钟学会 X」— AI 短视频化学习

**概念**：用短视频讲述 + 闯关验证的模式，适配碎片化学习。

- AI 将知识点拆解为 1 分钟的短视频脚本 + 自动配音
- 看完视频 → 立刻 3 题闯关验证 → 分享「一分钟学会XXX」成就卡
- 覆盖场景：通勤、等餐、上厕所——"随手学一点"
- **走红潜力**：短视频是当前最高频的内容消费形式，降低学习门槛

### 🌟 亮点 4：「知识星球」— 可视化知识图谱

**概念**：用户学习的每一个知识点都变成一颗"星球"，关联知识点形成星系。

- 每个用户拥有自己的"知识星球"，学得越多、星球越亮
- 知识点之间有引力连线（如 "Python" 关联到 "机器学习"）
- AI 推荐"你的知识星系还缺一颗 X 星，要探索吗？"
- **走红潜力**：高度可视化 + 收藏/养成心理，用户愿意截图分享

### 🌟 亮点 5：「AI 模拟面试」— 垂直场景深度方案

**概念**：针对高价值场景（求职/考研复试），提供沉浸式 AI 模拟面试。

- AI 扮演面试官，基于岗位 JD 和常见面试题进行提问
- 用户语音/文字回答 → AI 评估答案质量 → 给出改进建议
- 模拟面试结束后生成评分报告（沟通能力、专业度、逻辑性等维度）
- **变现潜力**：高频刚需 + 高付费意愿

### 🌟 亮点 6：「拖延症克星」— 游戏化学习习惯养成

**概念**：用游戏的机制对付学习的拖延。

- **每日能量条**：每天有固定的"学习能量值"，用完即止（饥饿营销）
- **学习连胜**：连续学习天数形成 streak，有皇家匹配特效，断签有"续命卡"
- **Boss 战**：每完成一个知识模块，解锁一个 Boss 战（综合大考）
- **宝箱掉落**：随机掉落"双倍经验卡""免答卡""换题卡"等道具
- **走红潜力**：游戏化是提升留存的最有效手段，参考 Duolingo 的成功

### 🌟 亮点 7：「知识胶囊」— AI 时间胶囊

**概念**：用户设定一个学习目标，AI 定期推送学习内容。

- 比如"30 天学会 Python 基础" → AI 每天推送一个知识点 + 5 道题
- 每 7 天一次阶段性测验，30 天后大考
- 学完开启"时间胶囊"——看到第一天 vs 第三十天的能力变化
- **走红潜力**：仪式感强，用户愿意分享"毕业证书"到朋友圈

### 🌟 亮点 8：「万物可题」— 多模态输入零门槛

**概念**：真正做到"看到什么，就能学什么"。

- **拍课本** → OCR+AI → 自动出题
- **拍路牌/菜单/说明书** → AI 识别 → 趣味问答
- **录音一段话**（如会议内容）→ AI 提取要点 → 出题验证理解
- **拍一道你不会的题** → AI 不仅解答，还生成 5 道同类题帮你练习
- **走红潜力**：极低的使用门槛，"拍一拍就能学"本身就有传播价值

## 5. 用户体验流程

### 5.1 核心用户旅程（Happy Path）

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 进入小程序 │───→│  输入知识  │───→│  AI 出题   │───→│  闯关答题  │───→│  结果复盘  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
     │               │               │               │               │
     │          · 打字输入         │          · 逐题作答         │          · 成绩单
     │          · 粘贴链接    等待3-10秒    · 即时反馈         │          · 知识总结
     │          · 上传文档    出题动画     · 连击特效         │          · 错题回顾
     │          · 语音输入         │          · 进度展示         │          · 分享海报
     │               │               │               │               │
     └───────────────┴───────────────┴───────────────┴───────────────┘
                                    │
                              ┌─────┴─────┐
                              │  个人中心   │
                              ├───────────┤
                              │ · 学习历史  │
                              │ · 错题本    │
                              │ · 排行榜    │
                              │ · 成就墙    │
                              └───────────┘
```

### 5.2 关键页面描述

#### 首页

- **搜索框（核心入口）**："今天想学什么？"
- **快捷入口**：上传文档、粘贴链接、语音输入、拍照识别
- **热门闯关推荐**：展示热门闯关游戏（如 "Python面试高频题"、"中国近代史10大事件"）
- **今日挑战**：每日一道精选趣味题
- **我的闯关**：未完成的闯关 + AI 推荐复习
- **底部导航**：首页、发现、发布、消息、我的

#### 答题页

- **顶部状态栏**：退出按钮、当前题号/总题数（如 第 3/10 题）、生命值显示
- **进度条**：实时展示答题进度
- **题目区域**：题号 + 题目内容 + 可选配图
- **选项区域**：4 个选项（A/B/C/D），点击后展示对错结果
- **知识讲解卡片**：答对/答错后展示知识要点与来源
- **连击特效**：连续答对时显示 combo 效果
- **操作按钮**：下一题

#### 通关结果页

- **结果展示**：得分（如 8/10）、正确率、用时
- **掌握度雷达图**：可视化知识子领域掌握情况
- **知识总结**：AI 生成的 3-5 条核心要点
- **错题回顾**：列出错题及详细解析入口
- **分享按钮**：一键生成精美海报，分享到朋友圈/群
- **再来一局**：重新挑战

## 附录：关键术语表

| 术语 | 说明 |
|------|------|
| **闯关** | 一次完整的答题过程，包含 N 道题目，用户逐题作答 |
| **知识领域** | 用户学习的知识范畴，如"Python编程""中国近代史" |
| **知识库** | B端功能，企业/个人上传的文档集合，作为出题素材 |
| **题目校验** | AI 二次验证题目和答案的准确性 |
| **连击（Combo）** | 用户连续答对题目数量的计数，用于增强游戏感 |
| **生命值** | 游戏化模式下用户的容错次数 |
