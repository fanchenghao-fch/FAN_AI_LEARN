知识要点

Python 中，列表(list)、字典(dict)、集合(set) 是可变类型，其内容可以修改。而 字符串(str)、元组(tuple)、整数(int) 是不可变类型，一旦创建就不能修改。

AI 知识要点总结
1 可变 vs 不可变类型：list/dict/set 可变，str/tuple/int 不可变。这是 Python 内存管理的核心概念。
2 def 关键字：Python 用最短的 "def" 定义函数，体现其简洁哲学。区别于 JS 的 function。
3 列表推导式：[x for x in range(10)] 是 Python 特有的高效写法，比循环更快更简洁。
4 异常处理：try/except/finally 结构确保代码健壮性，except 可以捕获多种异常类型。
